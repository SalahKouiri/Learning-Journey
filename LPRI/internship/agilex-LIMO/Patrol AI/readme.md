An autonomous industrial inspection system that bridges Edge and Cloud computing. The system utilizes a mobile robot running a "Split-Brain" architecture: leveraging 2D Lidar for zero-drift navigation and a 3D Voxel layer for spatial awareness. A localized computer vision model (YOLOv8) acts as the reflexes, detecting floor anomalies in real-time, while a multimodal LLM (Gemini) acts as the reasoning engine to classify the hazard level and instantly report detailed analytics to a centralized web-based fleet dashboard.
