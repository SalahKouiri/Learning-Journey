## Current Platform

Robot platform: **AgileX LIMO**

The LIMO robot is a mobile robotics platform capable of operating in multiple motion configurations, including:

- Ackermann steering  
- Four-wheel differential drive  
- Mecanum drive  

The robot integrates several components such as:

- Onboard computer  
- Depth camera  
- LiDAR  
- Wireless networking  
- ROS-based control stack
