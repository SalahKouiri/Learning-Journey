# Data Science Phase

This folder documents the **first phase of my personal AI learning roadmap**.

My long-term goal is to build strong foundations that will eventually lead me toward **AI research and a PhD**. To reach that goal, I designed a personal learning path structured in several stages:

1. Data Science  
2. Machine Learning  
3. Deep Learning  
4. AI Engineering  

Currently, I am in **Phase 1: Data Science**.

## Learning Approach

I am treating this phase as a **personal bootcamp**, using online courses, projects, and hands-on practice to develop strong data skills.

I mainly use **Coursera courses and specializations**, but I also plan to complement them with personal projects and additional resources when needed.

The idea is **not to follow everything strictly in order**, but to work on different topics simultaneously depending on what I need to learn or practice.

## Courses and Specializations

The main learning resources for this phase include:

- **IBM Data Science Professional Certificate**
- **Advanced Statistics for Data Science Specialization**
- **Data Visualization: Fundamentals to Interactive Storytelling Specialization**
- **Learn SQL Basics for Data Science Specialization**
- **Data Wrangling with Python Specialization**

These programs cover topics such as:

- Python for data analysis
- Statistics and probability for data science
- Data visualization
- SQL and databases
- Data cleaning and transformation
- Data analysis workflows
