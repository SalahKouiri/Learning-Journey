# IBM Data Science Professional Certificate

This folder documents my progress through the **IBM Data Science Professional Certificate** offered by IBM on Coursera.

The goal of this program is to build a strong foundation in **data science, data analysis, and machine learning using Python**, while developing practical skills through hands-on labs and real-world projects.

This course is part of my personal learning roadmap:

Data Science → Machine Learning → Deep Learning → AI Engineering

---

# Course Overview

The **IBM Data Science Professional Certificate** is a professional program designed to teach the core skills used by data scientists, including data analysis, visualization, machine learning, and working with data using Python and SQL.

The program includes multiple courses that guide learners through the **entire data science workflow**, from understanding the role of a data scientist to building machine learning models and completing a final capstone project.

Estimated completion time: **4–5 months (around 10 hours per week)**.

Key skills developed in this program:

- Python programming for data science
- Data wrangling and preprocessing
- Data visualization
- SQL and databases
- Exploratory data analysis
- Machine learning fundamentals
- Data storytelling and reporting
- Building portfolio-ready projects

---

# Course Curriculum

The certificate includes several courses covering the full data science workflow.

### 1. What is Data Science?
Introduction to the field of data science, career paths, and the role of data scientists.

### 2. Tools for Data Science
Overview of tools used in data science such as:

- Jupyter Notebooks
- Python
- R
- GitHub
- IBM Watson tools

### 3. Data Science Methodology
Understanding the **data science lifecycle**, including:

- Problem definition
- Data collection
- Data preparation
- Modeling
- Evaluation
- Deployment

### 4. Python for Data Science, AI & Development
Learning Python fundamentals and important libraries such as:

- NumPy
- Pandas
- APIs
- Data handling techniques

### 5. Python Project for Data Science
Hands-on project applying Python to analyze real datasets.

### 6. Databases and SQL for Data Science
Using SQL to:

- Query databases
- Extract data
- Integrate SQL with Python workflows

### 7. Data Analysis with Python
Techniques for:

- Exploratory data analysis
- Statistical analysis
- Data cleaning and transformation

### 8. Data Visualization with Python
Creating visualizations using:

- Matplotlib
- Seaborn
- Interactive visualization tools

### 9. Machine Learning with Python
Introduction to machine learning concepts such as:

- Classification
- Regression
- Clustering
- Model evaluation

### 10. Applied Data Science Capstone
A final project where learners apply the entire data science pipeline to solve a real-world problem.
