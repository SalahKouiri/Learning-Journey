# Open Source Tools for Data Science

This folder contains my notes, summaries, and key takeaways from the **Open Source Tools for Data Science** course on Coursera.

## Course Link
[Open Source Tools for Data Science – Coursera](https://www.coursera.org/programs/2ap-emsi-casablanca-orangers-g1-fm2n7/learn/open-source-tools-for-data-science?authProvider=emsi)

## About the Course
This course introduces the main open-source tools, languages, environments, and workflows used in data science. It also covers practical platforms and collaboration tools that are essential for modern data science work.

## Course Content

### 1. Overview of Data Science Tools
This module introduces the different categories of tools used by data scientists and gives examples of each. It also explores **Open Source**, **Cloud-based**, and **Commercial** tools for data science.

### 2. Languages of Data Science
This module presents the main programming languages used in data science, including:

- Python
- R
- SQL
- Java
- Scala
- C++
- JavaScript
- Julia

It also explains how to think about which language to learn first and how these languages are used in real data science workflows.

### 3. Packages, APIs, Data Sets, and Models
This part of the course focuses on important libraries and resources in data science. It includes:

- data science libraries and packages
- APIs and REST requests/responses
- open datasets
- machine learning models
- model and data asset platforms

### 4. Jupyter Notebooks and JupyterLab
This module introduces **Jupyter Notebook** and **JupyterLab** as important tools for recording, testing, and sharing data science work. It also covers:

- notebook kernels
- basic Jupyter architecture
- Anaconda Jupyter environments
- cloud-based notebook environments

### 5. RStudio & GitHub
This is one of the most practical parts of the course. It includes:

- introduction to **R** and **RStudio**
- R visualization packages
- creating charts with the **plot** function
- version control concepts
- Git and GitHub fundamentals
- repositories, commits, branches, pull requests, and merges
- collaborative coding workflows

### 6. Final Project and Submission
The course ends with a final project designed to apply the concepts and tools learned throughout the course. It also evaluates understanding of the main components of a data scientist’s toolkit.
