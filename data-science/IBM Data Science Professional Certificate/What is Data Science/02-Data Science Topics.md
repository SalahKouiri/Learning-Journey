# Big Data, Cloud Computing, and AI Fundamentals

## Digital Transformation
Digital transformation is the integration of digital technologies into all areas of a business, fundamentally changing how organizations operate and deliver value.

Example:  
The Houston Rockets NBA team used data analysis to improve their strategy by focusing on **two-point dunks and three-point shots**, which increased their chances of winning. This demonstrates how data-driven decisions can improve performance.

# Cloud Computing

Cloud computing is the delivery of computing resources (servers, storage, applications, networks) over the internet on a **pay-as-you-use basis**.

It allows organizations to:
- access resources remotely
- scale infrastructure when needed
- collaborate easily

Professor Haidar describes cloud computing as a way to **bypass the physical limitations of your personal computer**.

### Cloud Deployment Models
- Public cloud
- Private cloud
- Hybrid cloud

### Cloud Service Models
- **IaaS** – Infrastructure as a Service  
- **PaaS** – Platform as a Service  
- **SaaS** – Software as a Service  

### Key Characteristics of Cloud Computing
1. On-demand self-service  
2. Broad network access  
3. Resource pooling  
4. Rapid elasticity  
5. Measured service

# Big Data

Big Data refers to extremely large and complex datasets that cannot be processed using traditional database systems.

It is defined by the **Five V’s**:

| Characteristic | Description |
|---|---|
| Volume | Massive amount of data generated |
| Velocity | Speed at which data is produced |
| Variety | Different data formats (text, images, video, etc.) |
| Veracity | Accuracy and reliability of data |
| Value | Useful insights extracted from data |

Big data emerged from large-scale problems like **Google’s PageRank algorithm**, which required analyzing massive datasets efficiently.

---

# Big Data Processing

Big data systems use **distributed computing**.

Large datasets are:
- split into smaller parts
- distributed across thousands of computers
- processed in parallel

### MapReduce Concept
1. **Map** → each computer processes a piece of the data  
2. **Reduce** → results are combined into the final output  

This approach allows scalable analysis of massive datasets.

---

# Big Data Technologies

Several open-source technologies are commonly used for big data processing.

| Tool | Purpose |
|---|---|
| Hadoop | Distributed storage and processing framework |
| HDFS | Hadoop Distributed File System for storing large datasets |
| Hive | Data warehouse system for querying big data using SQL |
| Spark | Fast analytics engine for real-time processing |

Spark improves performance using **in-memory computing**.

# Data Mining

Data mining is the process of discovering patterns and insights in large datasets.

### Data Mining Process
1. Goal setting
2. Selecting data sources
3. Data preprocessing
4. Data transformation
5. Data mining (analysis)
6. Evaluation of results

The process is **iterative**, meaning results are refined through repeated analysis.

Skills required include:
- statistics

# Artificial Intelligence and Machine Learning

### Artificial Intelligence (AI)
AI is a field of computer science focused on building systems that mimic human intelligence.

### Machine Learning
Machine learning is a **subset of AI** where algorithms learn patterns from data and make predictions without explicit programming.

Common applications:
- recommendation systems (Netflix, Facebook)
- fraud detection
- predictive analytics
- classification and clustering

### Deep Learning
Deep learning is a subset of machine learning that uses **neural networks with multiple layers**.

These models are powerful for tasks like:
- speech recognition
- image classification
- natural language processing

Deep learning becomes more effective as **data volume increases**.

### Neural Networks
Neural networks are computational models inspired by the human brain.

They consist of:
- input layers
- hidden layers
- output layers

They learn patterns by adjusting internal weights during training.

---

# Generative AI

Generative AI is a subset of artificial intelligence that focuses on **creating new data rather than only analyzing existing data**.

Examples include generating:
- images
- text
- music
- code

It can also generate **synthetic data** to help train machine learning models when real data is limited.

Generative AI helps data scientists:
- discover deeper insights
- overcome data limitations
- improve data-driven decision making

# Statistical Methods

Example:
Predicting house price based on:
