## What is Data Science?

Data science is the field of exploring, manipulating, and analyzing data, and using data to answer questions or make recommendations.

Data science involves analyzing large amounts of data from different sources to discover insights and solve problems for organizations. Data scientists must ask the right questions, identify the data needed, and use different analytical methods to study that data. They then communicate their results through storytelling and visualization to help organizations make better decisions.

Because data science is not traditionally taught as a single discipline in universities, many data scientists come from different backgrounds such as engineering, statistics, mathematics, or physics.

According to Professor Haider, three important qualities for becoming a successful data scientist are:

- Curiosity
- Good judgement
- The ability to question and argue ideas

## Key Data Science Terms

| Term | Definition |
|-----|-----|
| Algorithms | A set of step-by-step instructions used to solve a problem or complete a task. |
| Model | A representation of relationships and patterns in data used to make predictions or analyze systems. |
| Outliers | Data points that appear far from other values in a dataset and may represent anomalies or errors. |
| Quantitative analysis | A method that uses mathematical and statistical techniques to analyze numerical data. |
| Structured data | Data organized in a clear format such as rows and columns in tables. |
| Unstructured data | Data that does not follow a predefined format, such as text, images, videos, or audio. |

## Data Science Tools and Technologies

| Term | Definition |
|-----|-----|
| CSV / TSV | File formats used to store tabular data where values are separated by commas or tabs. |
| Data file types | A specific file configuration designed to store data in a particular way. |
| Data format | The structure used to encode and store data inside a file. |
| Data visualization | Representing data visually (graphs, charts, etc.) to better understand patterns and trends. |
| Delimited text file | A text file where a special character separates each value. |
| XML (Extensible Markup Language) | A language used to store and transport structured data between systems. |
| Hadoop | An open-source framework used to store and process very large datasets across many computers. |
| JSON (JavaScript Object Notation) | A lightweight data format used for exchanging structured data between applications. |
| Jupyter notebooks | An environment used to write and share code, visualizations, and explanations in one document. |
| Nearest neighbor | A machine learning algorithm that predicts values based on similarity to other data points. |
| Neural networks | A deep learning model inspired by the human brain used to analyze complex data. |
| Pandas | A Python library used for data manipulation and analysis. |
| Python notebooks | Another name for Jupyter notebooks used to create interactive coding documents. |
| R | A programming language used for statistics, data analysis, and visualization. |
| Recommendation engine | A system that suggests products or content based on user behavior or preferences. |
| Regression | A statistical method used to study relationships between variables. |
| Tabular data | Data organized into rows and columns. |
| XLSX | The Microsoft Excel spreadsheet file format. |

## Summary

Data science is the study and analysis of large amounts of data in order to discover useful insights that help organizations make strategic decisions.

There are many paths to becoming a data scientist. Most involve mathematics, programming, and curiosity about data.

Successful data scientists are typically:
- Curious
- Judgemental
- Argumentative (able to question ideas and think critically)

Data science professionals are in high demand, and many jobs in this field offer high salaries for skilled workers.

The daily work of a data scientist can vary depending on the project they are working on.

Different algorithms and analytical techniques are used to extract insights from data.

Some important terms introduced in this lesson include:

- Outliers
- Model
- Algorithms
- JSON
- XML
- CSV
- Regression
