# Learning Journey

The goal of this repository is to keep a structured record of what I learn every day, including notes, experiments, courses, and small projects. It acts as both a **learning log** and a **technical archive** of my progress.

---

## Purpose

This repository serves several purposes:

- Track my daily progress and learning
- Document concepts, tools, and technologies I explore
- Store small experiments and mini-projects
- Organize resources and notes from courses
- Build a transparent record of my technical growth
